# jq_dict Raw Data

## pilot.json

Hand-authored seed for industry / concept / index / field entities (~53 records).
Static code suffixes (`.XSHG`, `.XSHE`, etc.) are merged at ingest from
`chunkers/jq_dict.py::STATIC_SUFFIXES`.

## full.json

Generated by the crawl script (no Chrome login required for industry/concept/index):

```bash
cd backend
uv run python scripts/crawl_jq_dict.py          # writes raw/full.json
uv run python scripts/ingest_jq_dict.py --reset # full ingest (all raw/*.json)
```

Sources:
- `help/api/plateData` — industry + concept (~1174)
- `data/dict/indexData` — index list (~732)
- `pilot.json` — field rows (help SPA has no static field tables in HTML)

## Eval datasets

Questions are generated from `raw/full.json` (codes validated against the crawl)::

```bash
cd backend
uv run python scripts/build_eval_jq_dict.py --mode all --force
uv run python scripts/eval_jq_dict.py --pilot   # 20 questions
uv run python scripts/eval_jq_dict.py           # 40 questions (pilot + extended)
```

## Regenerate (pilot only)

```bash
cd backend
.venv/bin/python scripts/ingest_jq_dict.py --pilot --reset
uv run python scripts/build_eval_jq_dict.py --mode pilot --force
uv run python scripts/eval_jq_dict.py --pilot
```
